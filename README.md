# Volibear - O Urso dos Mil Flagelos (Exportação GitHub)

Este diretório contém a versão puramente estática em **HTML, CSS e JavaScript Vanilla** do site imersivo do Volibear. Ela foi otimizada para o **GitHub** e **GitHub Pages**, garantindo que as animações, o canvas do raio e a sincronização do áudio nativo funcionem perfeitamente sem a necessidade de compiladores externos ou dependências pesadas.

## 📂 Estrutura de Arquivos

*   `index.html` — Estrutura semântica principal e views dinâmicas (Estilo com auxílio do Tailwind CSS).
*   `style.css` — Efeitos especiais personalizados de brilho roxo (*Urso dos Mil Flagelos*) e barras de rolagem customizadas.
*   `script.js` — Motor de raios em Canvas, efeitos de tremor (Camera shake) e gerenciador de áudio nativo.
*   `volibear_quotes.mp3` — **Importante:** Copie o arquivo de áudio `volibear_quotes.mp3` que você enviou (localizado na pasta `public` ou raiz do projeto) e coloque-o diretamente nesta mesma pasta do export antes de subir para o GitHub!

---

## ⚡ Como publicar no GitHub Pages

1. Crie um novo repositório no seu GitHub (ex: `site-volibear`).
2. Envie os arquivos desta pasta (`index.html`, `style.css`, `script.js` e o arquivo `volibear_quotes.mp3`) para a raiz do repositório.
3. Vá em **Settings** (Configurações) no seu repositório do GitHub.
4. No menu lateral esquerdo, clique em **Pages**.
5. Em **Build and deployment**, selecione a branch `main` (ou `master`) e a pasta `/ (root)`, e clique em **Save**.
6. Aguarde alguns minutos e o seu site magnífico de tempestades estará disponível online para todo o mundo!
